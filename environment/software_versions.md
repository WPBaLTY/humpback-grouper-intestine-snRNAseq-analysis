# Software Versions

## Exact Raw Reconstruction

Validated on 2026-07-11:

- R 4.6.1
- Seurat 5.3.0
- SeuratObject 5.2.0
- harmony 1.2.3
- dplyr 1.2.1
- ggplot2 4.0.3

The historical final global object was serialized by R 4.4.0. Its 102,036 cell-level resolution-0.4 cluster assignments were reproduced exactly under R 4.6.1 after restoring Seurat, SeuratObject, and harmony to the versions above.

The same locked versions were also used for the immune branch. Starting from the global `Leukocytes` subset generated by the eight-matrix rerun, the workflow reproduced the historical 10,440-cell initial immune object and the final 5,061-cell immune object exactly at the cell, cluster/annotation, sparse-count, and UMAP-coordinate levels.

Use `environment/install_exact_r_packages.R`. Source URLs and SHA256 hashes are in `environment/legacy_r_package_sources.csv`; the complete successful session is in `environment/raw_10x_rerun_sessionInfo.txt`.

`scripts/R_analysis/00_reconstruct_global_atlas_from_10x.R` rejects other core package versions by default because Seurat/harmony version drift produced different clusters during audit.

## Doublet Assessment

Validated on 2026-07-11 to 2026-07-12:

- R 4.6.1
- Seurat 5.5.1
- SeuratObject 5.4.0
- SingleCellExperiment 1.34.0
- scDblFinder 1.26.7
- BiocParallel 1.46.0
- dplyr 1.2.1

This separate environment was used only to read the locked global object, construct a `SingleCellExperiment` from raw RNA counts, and run scDblFinder separately by capture library. Install it with `environment/install_doublet_r_packages.R`. Seeds and model parameters are recorded in `audit/doublet_assessment/doublet_analysis_parameters.csv`, and the full session is in `audit/doublet_assessment/doublet_sessionInfo.txt`.

## Sequencing Processing

- Cell Ranger 5.0.0
- Custom reference based on the chromosome-level *Cromileptes altivelis* genome assembly described by Sun et al. (bioRxiv, 2020; BioProject PRJNA639378)
- Eight web summaries independently report `Single Cell 3' v3`, intronic-read counting, reference basename `Cal`, and Cell Ranger 5.0.0; report SHA256 values are in `metadata/cellranger_run_metadata.csv`.
- The common Cell Ranger feature table has 26,134 rows: 26,121 IDs close exactly to the project nuclear GFF3 and 13 IDs/symbols close to `NC_021614.1`.
- The merged Seurat analysis object has 23,740 features after per-library `CreateSeuratObject(min.cells = 3)` filtering.

The original combined `cal_MT.fa` and exact 2 kb 3' UTR-extended `cal.3utr_MT.gtf` were not recovered. Source fingerprints and the resulting FASTQ-to-matrix reproducibility boundary are documented in `REFERENCE_PROVENANCE_AUDIT.md`.
