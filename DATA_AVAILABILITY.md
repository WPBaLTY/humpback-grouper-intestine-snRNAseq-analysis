# Data availability map

| Content | Repository |
|---|---|
| Raw FASTQ files | GEO/SRA `GSE326285` |
| Per-library 10x matrices | GEO `GSE326285` |
| Processed metadata, UMAP, marker tables, figure source data, audit tables and integrated Seurat objects | Zenodo DOI `10.5281/zenodo.21481391` |
| Code, run order, validators and environment notes | GitHub |
| Final figure outputs and supplementary files | Zenodo and journal submission package |

During first-round Scientific Data review, all cited data must be anonymously downloadable. From the second review round onward, the formal repository records should be public.
